## 2. 演算
普通に×
- 宇宙船演算子
左辺が小さかったら−１
左辺が大きかったら１
同じだったら０
```
1 <=> 3 # -1
3 <=> 1 # 1
3 <=> 3 # 0

- 四捨五入
```
# 小数部をカット
p 1.4.round # 1
p 1.5.round #2
p -1.4 round # -1
p -1.5 round # -2

# 小数点以下　ｎ桁の四捨五入
