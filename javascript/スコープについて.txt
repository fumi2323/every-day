Javascriptのスコープについて

まず、スコープは変数が有効な範囲のこと。
どこからでもアクセスできるのはグローバル変数なので基本的にスコープの外にあるということになる。

その反対。同じスコープの中しか使えない変数はローカル変数という。


スコープは{}で囲ってもjsの場合スコープにならない。

でも、
function aaa(){
  //この中はスコープ内。ローカルになる。
  var test = "これはローカル変数";
  
  //ただし、varをつけないとグローバルになるので覚えておこう。
  test2 = "これはグローバル変数";
 
}
console.log(test); エラー
console.log(test2); "これはグローバル変数"　//と表示される
